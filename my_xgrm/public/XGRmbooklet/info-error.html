Section 5 INFO & ERROR

Info Buttons

Each user-request interface features an info toggle button labelled Show/Hide Info. This button contains information on how to use the interface, including details on input, output, and other relevant information. Additionally, there is an Example button that showcases example inputs and outputs. For example, the screenshot below displays the Show/Hide Info toggle button in the GElyser interface.
